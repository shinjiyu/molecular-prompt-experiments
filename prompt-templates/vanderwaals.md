# 范德华键模板（Self-Exploration - 并行探索）

## 核心思想

范德华键代表概念探索和弱桥接："探索新方向，寻找替代方案"。这种推理模式强调：
- 同时探索多条推理路径
- 对比不同方法的优劣
- 扩展推理空间

## 模板内容

```
【目标】同时探索两条不同的推理路径，对比结果

对于此问题，请同时构建两条独立的推理路径：

【路径 A】（常规/直接方法）
├─ 步骤A1: ...
├─ 步骤A2: ...
└─ 结论A: ...

【路径 B】（替代/逆向方法）
├─ 步骤B1: ...
├─ 步骤B2: ...
└─ 结论B: ...

【对比分析】
├─ 结论一致性: [A和B是否得到相同结果]
├─ 复杂度对比: [哪条路径更简洁]
├─ 可靠性对比: [哪条路径更不易出错]
└─ 最终选择: [选择哪条路径的结论，为什么]

如果两条路径结论不同，分析差异原因并解决。

现在请解决以下问题：
{question}
```

## 适用场景

- ✅ 开放式问题（有多种解决方案）
- ✅ 策略规划（需要权衡不同方案）
- ✅ 创新性问题（需要探索新思路）
- ✅ 不确定性高的问题

## 不适用场景

- ❌ 有明确唯一答案的问题
- ❌ 时间/资源受限的场景
- ❌ 简单线性推理问题

## 预期效果

- **优势**：探索广度大，创新性强
- **劣势**：可能发散过度，效率低
- **键分布**：范德华键占比 10-20%

## 使用建议

1. 在不确定性高的节点使用探索
2. 及时收敛，避免无限探索
3. 记录所有探索路径，供后续参考

## 探索策略变体

### 1. 双路径对比
- 路径 A：常规方法
- 路径 B：逆向/替代方法

### 2. 多假设验证
- 列出所有可能的假设
- 逐一验证
- 选择最合理的

### 3. 竞争性推理
- 多个策略竞争
- 相互攻击弱点
- 裁判选出最优
